# community-fresh-group-buy-admin

## 答辩演示步骤

1. 启动后端服务（Spring Boot）与数据库（MySQL）。
2. 启动管理端前端：
   - `pnpm install`
   - `pnpm dev`
3. 登录管理端（开发登录）。
4. 打开“仪表盘”，演示统计卡片与快捷跳转。
5. 进入“订单管理”：
   - 按状态筛选
   - 关键词搜索
   - 分页切换
   - 更新订单状态（待成团 -> 已成团 -> 已取货）
   - 查看订单详情抽屉
   - 导出当前筛选结果 CSV
6. 进入“用户管理”：
   - 查看用户列表
   - 关键词搜索（ID/昵称/手机号）
7. 异常场景演示：
   - 非法状态流转被拦截
   - 接口失败时错误提示可见
